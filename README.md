# Currency Converter App

This is a Currency Converter web application built using **HTML**, **CSS**, and **JavaScript**.  
The app connects to a real-time exchange rate API to provide accurate and up-to-date currency conversions.

## Features

- Convert between multiple currencies  
- Fetch live exchange rates from an external API  
- Simple and user-friendly interface  
- Fast and responsive design  

## Purpose

This project is created for learning purposes and to practice working with APIs and asynchronous JavaScript.
